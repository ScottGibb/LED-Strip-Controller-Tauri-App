## Dev Environments

```bash
  cd led-strip-controller-tauri-app
  bun install
```

For Desktop development, run:

```bash
  bun run tauri dev
```

For Android development, run:

```bash  
  bun run tauri android dev
```

For iOS development, run:

```bash
  bun run tauri ios dev
```
